# lab #2 html forms and get/post requests

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AlexandrPristupa/lab.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Open termional and run node server
    ```sh
   npm run start
   ```
4. Open browser 
    ```sh
   localhost:3000
   ```
